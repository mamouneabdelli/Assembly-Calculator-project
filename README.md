## 🧮 Assembly Calculator

A simple and interactive calculator built using **8086 Assembly Language**.
This project demonstrates low-level programming concepts such as user input handling, arithmetic operations, and control flow using DOS interrupts.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 📋 Menu-driven interface
* 🔁 Loop for multiple operations
* ⚠️ Basic error handling (e.g., division by zero)

---

## 🛠️ Technologies Used

* 8086 Assembly Language
* DOS Interrupts (`INT 21h`)
* Compatible with **DOSBox**, **TASM**, or **MASM**

## ▶️ How to Run

1. Install DOSBox
2. Assemble the program:

   ```
   tasm calculator.asm
   ```
3. Link the object file:

   ```
   tlink calculator.obj
   ```
4. Run the program:

   ```
   calculator.exe
   ```
